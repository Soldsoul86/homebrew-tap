# homebrew-tap

Homebrew formulas for [Soldsoul86's](https://github.com/Soldsoul86) tools.

```
brew install Soldsoul86/tap/checkpoint
brew install Soldsoul86/tap/loopkill
brew install Soldsoul86/tap/ctxmeter
brew install Soldsoul86/tap/again
```

Formulas in `Formula/` are published automatically by each tool's release
pipeline in [Soldsoul86/AAA](https://github.com/Soldsoul86/AAA) — nothing
in this repo is meant to be edited by hand.
